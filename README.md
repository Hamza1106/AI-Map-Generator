# 🚀 Getting Started

## Prerequisites

- Python 3.10+
- pip

---

## Installation

Clone the repository

```bash
git clone https://github.com/Hamza1106/ai-world-generator.git
```

Navigate to the project

```bash
cd ai-world-generator
```

Install dependencies

```bash
py -m pip install -r requirements.txt
```

---

## Run the Project

Launch the world generator

```bash
python game.py
```

The engine will generate a unique AI-powered procedural world ready for real-time exploration.
